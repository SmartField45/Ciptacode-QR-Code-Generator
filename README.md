# Ciptacode QR Code Generator




A multifunctional web application based on Python and Streamlit to interactively generate QR Codes. This application allows users to generate QR Codes from various types of data, customize visuals, and download the results directly.



---



## Main Features



- **Text and URL:** Convert website links or plain text into QR Codes.

- **Wi-Fi Access:** Create QR Codes that allow users to connect to a Wi-Fi network automatically when scanned.

- **Contact Card (vCard):** Generate QR Codes containing contact information (Name, Phone, Email) that can be directly saved to the user's phonebook.

- **WhatsApp Message:** Create QR Codes that will direct the scanner straight to the WhatsApp application along with an automated draft message.

- **Visual Customization:** Adjust the main QR Code color, background color, and image resolution size.

- **Logo Insertion:** Upload a custom image or logo to be placed right in the center of the QR Code without breaking the scanning functionality (using a high error correction level).



## System Requirements



To run this application on your local computer, ensure you have installed:

- Python 3.7 or a newer version.

- The Python libraries listed in the `requirements.txt` file.



## Installation and How to Run Locally



1. Download or clone this repository to your computer.

2. Open a terminal or command prompt, then navigate to this project folder.

3. Install all required libraries by running the following command:

```bash
pip install -r requirements.txt
```
